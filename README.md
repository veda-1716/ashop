


- **Clone/Download the repo**
- **composer install**
- **copy .env.example .env**
- **php artisan key:generate**
- **php artisan migrate**
- **php artisan db:seed**
- **php artisan serve**
- **[http://127.0.0.1:8000/](http://127.0.0.1:8000/)**
